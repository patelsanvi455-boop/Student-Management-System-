Student Management System

1. Introduction
This is a C++ Student Management System project of C++.
It manages student details like Roll Number, Name, and Marks.

2. Objectives
- Stores Student Details 
- Display all student records
- Search for a student by roll number
- Learn basic C++ OOP
- Make a simple menu-driven program 

3. System Design
- Class is used to store student data
- Array of Objects is also used to store multiple students
- Menu Driven Program using switch-case
- Search Functionality is used to find students
